# Python Task Manager

A simple command-line Task Manager application built using Python.

## Features
- Add tasks
- View tasks
- Update tasks
- Delete tasks
- Persistent storage using JSON

## Concepts Used
- Python OOP
- File handling
- CRUD operations
- Error handling

## How to Run
```bash
python task_manager.py
## Project Purpose

This project was built to strengthen Python backend fundamentals such as
object-oriented programming, file handling, validation, and CRUD operations.
It reflects real-world backend logic commonly used in frameworks like Frappe
and Django.
